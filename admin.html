Importar Produtos em Massa

📤 Importar via Planilha (CSV)

Faça upload de um arquivo CSV com seus produtos. A planilha deve ter as seguintes colunas:

nome, preco, unidade, categoria, emoji, estoque

Exemplo: Arroz Branco,12.90,kg,mercearia,🍚,50

🔗 Integração via API

Se seu sistema atual tem uma API, você pode integrar automaticamente usando este endpoint:

POST http://localhost:3000/api/admin/produtos/bulk 
{
  "produtos": [
    {
      "nome": "Arroz Branco",
      "preco": 12.90,
      "unidade": "kg",
      "categoria": "mercearia",
      "emoji": "🍚",
      "estoque": 50
    }
  ]
}
