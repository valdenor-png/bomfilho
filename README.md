# Bom Filho Web

Bom Filho Web é um sistema web de supermercado com foco em experiência de compra simples, rápida e funcional. O projeto reúne um frontend em HTML, CSS e JavaScript com um backend em Node.js e Express, integrado ao MySQL para persistência de dados.

A aplicação cobre o fluxo principal de um e-commerce local: autenticação de usuários, catálogo de produtos, carrinho, checkout, criação de pedidos, histórico, cupons, avaliações e rastreamento. A estrutura foi organizada para facilitar manutenção e evolução contínua, com separação entre interface, regras de negócio e API.

O objetivo do projeto é servir como base prática para operação e crescimento de um delivery de supermercado, com arquitetura enxuta, código direto e foco em estabilidade no uso real.
